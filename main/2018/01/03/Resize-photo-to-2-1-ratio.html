Photos to Facebook 360

A tutorial of converting mulitple photos to spherical 360 photos for Facebook.

1. Combine 2. Rezise 3. Tag

This tutorial demonstrate the steps of:

  1. Resizing your image to 2:1 aspect ratio
  2. Reduce the photo dimension

This is one of the requirments for Facebook to detect your photo is a spherical 360 photos.

Using GIMP (Linux / macOS / Windows)

  1. Image ➜ Scale Image…
  2. Set 6000 for width
  3. Click “Scale”
  4. Image ➜ Canvas size…
  5. Set 3000 for height (6000 / 2 = 3000)
  6. Hit Tab, let the canvas preview update
  7. In the canvas preview: drag the image down to bottom
  8. Click “Resize”
  9. File ➜ Export As… (Save as a new photo)
